# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- PowerShell module for Windows with cross-platform support
- System statistics display (CPU, RAM, Disk)
- Nerd Font icon support
- JSON configuration system
- Module status detection (Oh My Posh, PSReadLine, Terminal-Icons)
- Progress bars with color-coded percentages
- Cross-platform installer script
- GitHub Actions CI/CD workflows
- Comprehensive documentation

### Supported Platforms
- Windows 11/10 (PowerShell 7+)
- macOS (PowerShell 7+) - experimental
- Linux (PowerShell 7+) - experimental

### Features
- Real-time CPU load monitoring
- RAM usage with frequency detection
- Disk usage statistics
- Terminal/process count
- System uptime display
- Customizable icons and colors
- Theme support (default config)

## [0.1.0] - 2025-01-XX

### Added
- Project initialization
- Basic module structure
- Mock implementations for Zsh and Bash
